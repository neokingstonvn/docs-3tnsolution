1. Thiết Kế Logic RESTful APIs

Tiêu Chí Kỹ Thuật Giải Thích & Lợi Ích Lâu Dài Cho Dự Án
Phản hồi HTTP 202 Accepted Vì tác vụ xử lý AI tách nền và render tốn thời gian (lên đến 60s), API tạo phối cảnh sẽ trả về HTTP 202 ngay lập tức cùng với ID và thời gian chờ dự kiến, tránh làm nghẽn kết nối của Client.
Mô hình Polling / WebSocket Client sẽ gửi request polling hoặc kết nối socket để nhận cập nhật trạng thái realtime từ hàng đợi AI giúp tối ưu trải nghiệm.
Bảo mật & Phân quyền Mỗi API đều có middleware JWT xác thực. Admin mới có quyền POST/DELETE Template, Merchant chỉ có quyền CRUD trên ảnh sản phẩm và phối cảnh của chính họ.

3. Danh Sách RESTful API Endpoints

Phương thức Đường dẫn (Endpoint) Quyền truy cập Mục tiêu & Chức năng xử lý chính
POST `/api/auth/login` Public Xác thực thông tin đăng nhập và trả về mã JWT Token.
GET `/api/templates` All Roles Truy xuất danh sách template mẫu (hỗ trợ lọc theo tỷ lệ 1:1, 16:9).
POST `/api/templates` Admin Tải lên template thiết kế mới và lưu thông số Safe Zone.
DELETE `/api/templates/:id` Admin Xóa template mẫu ra khỏi cơ sở dữ liệu.
POST `/api/products` Merchant Đăng ký sản phẩm mới (Mã SKU định danh, tên sản phẩm).
GET `/api/products` Merchant Xem danh mục sản phẩm (các SKU) đã đăng ký.
POST `/api/products/:productId/images` Merchant Tải lên nhiều ảnh thô góc chụp cho mã sản phẩm, tự động gọi API AI tách nền và liên kết ảnh trong suốt.
POST `/api/compositions` Merchant Yêu cầu phối cảnh ảnh sản phẩm (đẩy tác vụ vào hàng đợi AI).
GET `/api/compositions/:id` Merchant Kiểm tra tiến trình và nhận link ảnh render cuối cùng.

5. Phân Công & Hoàn Thành

Thành viên Vai trò chính Tiêu chí hoàn thành (Definition of Done)
Backend Dev NestJS, CSDL, AI SDK Tách nền chính xác, trả API 202 bất đồng bộ, đẩy job vào hàng đợi thành công. Tối ưu thời gian chạy dưới 60s.
Frontend Dev React UI, Slider, Queue UI Tải ảnh mượt mà, hiển thị thanh chờ hàng đợi và loading trạng thái chuẩn xác, xuất bản và download ảnh độ phân giải gốc.
DevOps Redis, S3 Storage, CI/CD Thiết lập S3/MinIO và Redis ổn định, tự động dọn dẹp cache sau 24h, cấu hình server AI chịu tải tốt.
