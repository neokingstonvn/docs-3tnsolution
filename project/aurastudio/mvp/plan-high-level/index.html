1. Phân Tích Logic Nghiệp Vụ Cốt Lõi

Tiêu Chí Nghiệp Vụ Giải Thích & Lợi Ích Lâu Dài Cho Dự Án
Tại sao chọn PostgreSQL? PostgreSQL cung cấp hiệu năng truy vấn cao, hỗ trợ kiểu dữ liệu JSONB linh hoạt (phù hợp để lưu thông số coordinates bổ sung của safe zone) và tính toàn vẹn dữ liệu cực kỳ tốt.
Cấu hình Safe Zone động MVP thiết kế hệ thống tọa độ Safe Zone động (x, y, w, h) dưới dạng tỷ lệ phần trăm so với kích thước gốc của template. Giúp ảnh sản phẩm tự động co giãn vừa vặn trên mọi kích thước ảnh render.
Xếp hàng FIFO xử lý AI Giúp hệ thống tránh bị quá tải tài nguyên GPU khi nhiều người dùng gửi yêu cầu cùng lúc. Đảm bảo tính ổn định và cung cấp thông tin thời gian chờ dự kiến thực tế cho người dùng.
Phân lớp Layer rõ ràng Đảm bảo tính mỹ thuật của ảnh sản phẩm (chủ thể nằm dưới logo, text trang trí nhưng nằm trên phông nền template).

3. Database Schema (PostgreSQL DDL)

Tên bảng Thực thể chính Mục tiêu lưu vết nghiệp vụ
users User Lưu trữ thông tin tài khoản người dùng và gán vai trò quyền hạn (Admin, Merchant).
templates Template Lưu trữ tệp template thiết kế và các bối cảnh hình mẫu con.
products Product Quản lý danh mục sản phẩm của Merchant theo Mã sản phẩm (SKU).
product_images ProductImage Quản lý danh sách ảnh góc chụp thô và ảnh đã tách nền AI tương ứng với từng mã sản phẩm (SKU).
compositions Composition Lưu vết quá trình phối cảnh sản phẩm, thông số co giãn, dịch chuyển và tệp ảnh render cuối cùng.
processing_queues QueueTask Điều phối hàng đợi xử lý AI khi hệ thống bị quá tải.
audit_logs AuditLog Lưu trữ nhật ký thao tác nghiệp vụ và bảo mật. 

-- PostgreSQL DDL Schema for AuraStudio
-- (Chi tiết mã SQL được đồng bộ tại .agents/output/db_schema.sql)
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(255) UNIQUE NOT NULL,
    full_name VARCHAR(100) NOT NULL,
    role VARCHAR(50) NOT NULL CHECK (role IN ('Admin', 'Merchant')),
    created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
);

5. Stack Kỹ Thuật Lựa Chọn

Thành phần Công nghệ đề xuất Mô tả vai trò kỹ thuật chính
Frontend ReactJS + Tailwind CSS Xây dựng giao diện nhanh, linh hoạt, hỗ trợ slider và drag-drop mượt mà.
Backend Framework NestJS (TypeScript) Framework cấu trúc tốt, hỗ trợ Dependency Injection chuẩn cho Clean Architecture.
Database PostgreSQL + Prisma ORM Hệ quản trị CSDL quan hệ tin cậy, lưu trữ JSONB linh hoạt, kiểm soát kiểu dữ liệu tốt.
Queue System Redis (BullMQ) Quản lý hàng đợi tác vụ AI tách nền và render phối cảnh bất đồng bộ.
Object Storage AWS S3 / MinIO Lưu trữ các tệp ảnh thô ban đầu và ảnh thành phẩm độ phân giải cao.
