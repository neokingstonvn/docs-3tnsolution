# Prototype Directory - Task Kingston

Thư mục này chứa mã nguồn bản chạy thử nghiệm UAT (Prototype) của dự án **Task Kingston**, sử dụng HTML/CSS/JS thuần và mock cơ sở dữ liệu qua `localStorage`.

Các file dự kiến:
- `index.html` (Cổng đăng nhập và phân quyền Gateway Portal)
- `db.js` (Module điều phối dữ liệu giả lập và đồng bộ localStorage)
- `style.css` (Giao diện tối màu bóng bẩy v2)
- Các file view con cho từng Role (Admin, PM, Staff)

*Sẽ được sinh tự động bởi Agent `/prototype-generator`.*
