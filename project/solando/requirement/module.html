📁 Cấu Trúc Thư Mục src/

src/
├── domain/                    // Thực thể cốt lõi
│   ├── entities/              // Dealer, Product, Order,...
│   └── repositories/          // Interface cổng kết nối CSDL
├── use-cases/                 // Logic nghiệp vụ chi tiết
│   ├── authentication/        // LoginUseCase.ts
│   ├── master-data/           // ManageSettingsUseCase.ts
│   ├── dealer/                // CheckCreditLimitUseCase.ts
│   ├── quotation/             // CalculateMarginUseCase.ts
│   ├── billing/               // ProcessVietQRPaymentUseCase.ts
│   └── warranty/              // ActivateWarrantyViaQRUseCase.ts
├── infrastructure/            // Framework, CSDL, Webhook ngoài
│   ├── database/              // Prisma ORM, Repo implementation
│   ├── http/                  // NestJS Controllers
│   └── payment/               // VietQR webhook receiver

📦 9 Module Thực Tế Cần Phát Triển 🛠️

Tech Module 1

🔑 Xác Thực & Phân Quyền

Đăng nhập phân quyền Admin, Kinh doanh, Kho & Đại lý.

Entities (Domain)

User, Role, Permission, UserSession

Use Cases

LoginUseCase, ValidateUserSession, AssignRBACPermissions

Infra

Bcrypt mã hóa, Passport JWT, NestJS Guards.

Tech Module 2

⚙️ Dữ Liệu Dùng Chung

Danh mục dùng chung hệ thống, cài đặt cấu hình.

Entities (Domain)

Branch (kho), Currency (tiền tệ), SystemConfig

Use Cases

GetCategoriesList, UpdateSystemConfigParameter

Infra

Prisma database client, global config cache.

Biz Module 1

🏢 Đại Lý & Công Nợ

Quản lý hạn mức công nợ & phân hạng Đại lý.

Entities (Domain)

Dealer (Tier 1/2/Lẻ), EndUser (Khách cuối)

Use Cases

CheckCreditLimitUseCase, CreateDealerUseCase

Infra & Rules

Rule: Chặn cứng (Hard Block) tạo đơn mới khi nợ + đơn mới vượt hạn mức.

Biz Module 2

🏷️ Sản Phẩm & BOM

Danh mục SKU và combo kỹ thuật Solar.

Entities (Domain)

Product (SKU, bảo hành SKU), SolutionKit (Kit BOM)

Use Cases

RegisterProductUseCase, CreateSolutionKitUseCase

Infra

AWS S3 file lưu CO/CQ, Catalog tài liệu kỹ thuật.

Biz Module 3

📋 Báo Giá & Pricing

Pricing engine theo Tier và tính margin theo giá lô.

Entities (Domain)

Quotation, QuotationItem, PricingPolicy

Use Cases

CalculateMarginUseCase, ApproveQuotationUseCase

Infra & Rules

Rule: Kích hoạt workflow duyệt giá lên Admin khi Margin % < 10%.

Biz Module 4

🤝 Đơn Hàng (Orders)

Quản lý vòng đời đơn hàng bán cho đại lý.

Entities (Domain)

Order, OrderItem, OrderStatusHistory

Use Cases

CreateOrderFromQuotationUseCase, UpdateOrderStatus

Infra

Cơ chế giao dịch an toàn (DB Transactions) của Prisma.

Biz Module 5

💳 Hóa Đơn & VietQR

Xuất hóa đơn, sinh VietQR động & gạch nợ tự động.

Entities (Domain)

Invoice, PaymentTransaction

Use Cases

GenerateVietQRPayload, ProcessVietQRPaymentWebhook

Infra & Rules

Infra: SePay/Casso Webhook. Rule: Gạch nợ công nợ đại lý tức thì trong 3 giây.

Biz Module 6

📦 Kho & Quét Serial

Lô hàng (Batch), giá vốn COGS và quét Serial Number.

Entities (Domain)

Batch (Lot), Serial (In Stock, Shipped, Activated)

Use Cases

ImportBatchUseCase, ScanSerialExportUseCase

Infra

Ứng dụng quét camera HTML5 PWA trên điện thoại.

Biz Module 7

🛠️ Bảo Hành & Sự Cố

Kích hoạt bảo hành qua QR công cộng và quản lý Ticket.

Entities (Domain)

ActivatedAsset, SupportTicket

Use Cases

ActivateWarrantyViaQRUseCase, CreateSupportTicket

Infra

Zalo ZNS / SMS API thông báo kích hoạt thành công.
